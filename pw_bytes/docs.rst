---------
pw_bytes
---------
pw_bytes is a collection of utilities for manipulating binary data.

Compatibility
=============
C++17

Dependencies
============
* ``pw_preprocessor``
* ``pw_status``
* ``pw_span``

Features
========

pw::ByteBuilder
-----------------
ByteBuilder is a utility class which facilitates the creation and
building of formatted bytes in a fixed-size buffer.
